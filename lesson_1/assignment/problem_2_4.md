__Задача 2.4__

Какие из нижеперечисленных векторов
образуют ортонормированный базис
в трёхмерном действительном пространстве

1. `(1, 0, 0), (0, 0, 1)`;
2. `(1 / sqrt(2), -1 / sqrt(2), 0), (1 / sqrt(2), 1 / sqrt(2), 0), (0, 0, 1)`;
3. `(1 / 2, -1 / 2, 0), (0, 1 / 2, 1 / 2), (0, 0, 1)`;
4. `(1, 0, 0), (0, 1, 0), (0, 0, 1)`?

__Ответ:__

1. Не образуют, так как
не для всех векторов трёхмерного пространства
существует представление
в виде линейной комбинации этих векторов,
например, `(0, 1, 0)`.

2. Образуют в силу определения.

3. Образуют ортогональный,
но не нормированный.

4. Образуют в силу определения.
